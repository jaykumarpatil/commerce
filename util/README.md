# Module: util

Purpose:
- Shared utilities used across modules, including common exception handling, helpers, and base utilities.

What to expect:
- Helper classes, base exceptions, and lightweight utilities used by multiple services.
- No business logic; just generic utilities.

How to work with it:
- Build: `./gradlew build` from repo root.
- Tests: `./gradlew test`.
- For module usage, refer to the consuming module README.

Notes:
- Java 17 / Spring Boot 3.x stack is used across the project.
