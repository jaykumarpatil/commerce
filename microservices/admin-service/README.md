# Module: microservices/admin-service

Purpose:
- Administrative tooling and helpers for managing the microservice ecosystem (monitoring, maintenance, config).

How to run:
- Build: `./gradlew build` from repo root.
- Run: `./gradlew :microservices:admin-service:bootRun`.
- Tests: `./gradlew :microservices:admin-service:test`.

Notes:
- Not customer-facing; supports governance and ops tasks.
