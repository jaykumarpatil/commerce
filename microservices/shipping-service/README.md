# Module: microservices/shipping-service

Purpose:
- Coordinates shipment and fulfillment for orders.

How to run:
- Build: `./gradlew build` from repo root.
- Run: `./gradlew :microservices:shipping-service:bootRun`.
- Tests: `./gradlew :microservices:shipping-service:test`.

Notes:
- Works with order and payment services to ensure successful delivery.
