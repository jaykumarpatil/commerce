# Module: microservices/user-service

Purpose:
- Manages user accounts, authentication metadata, and profile information used across the system.

How to run:
- Build: `./gradlew build` from repo root.
- Run: `./gradlew :microservices:user-service:bootRun`.
- Tests: `./gradlew :microservices:user-service:test`.

Notes:
- Integrates with the auth domain for authentication and authorization flows.
