# Module: microservices/inventory-service

Purpose:
- Tracks product stock levels and availability across warehouses or fulfillment centers.

How to run:
- Build: `./gradlew build` from repo root.
- Run: `./gradlew :microservices:inventory-service:bootRun`.
- Tests: `./gradlew :microservices:inventory-service:test`.

Notes:
- Critical for planning fulfillment and detecting stockouts.
